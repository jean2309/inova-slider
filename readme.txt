== Descrição ==

Plugin desenvolvido para facilitar a criação de slider através do editor gutenberg.
Através do painel é possivel fazer algumas configurações no slider.
1 - Através do painel da para selecionar a quantidade desejada de imagens, mudar a ordem e remover imagens.
2 - Através do painel da para selecionar a quantidade de imagens que será exibida limitado de 1 a 6 imagens por slider.
3 - Através do painel da para deixar ou não o slider em modo loop infinito.


== Desenvolvimento ==

O projeto foi criado usando padrões da documentação: Codigo para criação do projeto: npx @wordpress/create-block@latest inova-slider --variant dynamic
Para o desenvolvimento do plugin, foi utilizado as tecnologias JQuery, React e PHP. 
Um bugs foi corrigido através de IA.
Bug - Espaço em branco ao deixar o slider em loop infinito.



== Instalação ==

Instalar pacotes
npm install

Iniciar projeto
npm run start

Fim
